# vivainsights: Analyze and Visualize Data from 'Microsoft Viva Insights'

Provides a versatile range of functions, including exploratory data
analysis, time-series analysis, organizational network analysis, and
data validation, whilst at the same time implements a set of best
practices in analyzing and visualizing data specific to 'Microsoft Viva
Insights'.
